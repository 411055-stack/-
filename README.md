# 📚 背單字卡片 - Flashcard App

一個現代化的英文單字學習應用，採用互動式翻頁卡片設計。使用純 HTML、CSS 和 JavaScript 開發，無需任何依賴或框架。

## ✨ 功能特性

- **3D 翻頁效果**：點擊卡片或按鈕可產生流暢的翻頁動畫
- **完整單字資訊**：
  - 英文單字
  - 中文翻譯
  - 詞性（名詞、動詞、形容詞等）
  - 音標（國際音標）
- **鍵盤快捷鍵**：
  - `←` 上一個單字
  - `→` 下一個單字
  - `空白鍵` 翻轉卡片
- **進度顯示**：實時顯示當前位置（X / 總數）
- **循環播放**：到達最後一個單字後會循環回到第一個
- **響應式設計**：適配桌面、平板和手機

## 🚀 如何使用

### 方式 1：直接在瀏覽器開啟
1. 用瀏覽器打開 `index.html` 檔案
2. 即可開始學習

### 方式 2：使用 Live Server（推薦）
如果使用 VS Code：
1. 安裝 **Live Server** 擴展
2. 右鍵點擊 `index.html` → **Open with Live Server**
3. 自動在瀏覽器打開

## 📖 操作說明

| 操作 | 效果 |
|------|------|
| 點擊卡片 | 翻轉卡片 |
| 按「🔄 翻轉卡片」按鈕 | 翻轉卡片 |
| 按「← 上一個」按鈕 | 切換到上一個單字 |
| 按「下一個 →」按鈕 | 切換到下一個單字 |
| 按左箭頭鍵 `←` | 切換到上一個單字 |
| 按右箭頭鍵 `→` | 切換到下一個單字 |
| 按空白鍵 `Space` | 翻轉卡片 |

## 💾 單字庫

應用內建 10 個常用高級英文單字，包括：
- Serendipity（巧合；幸運的發現）
- Ephemeral（短暫的）
- Ubiquitous（無所不在的）
- 和其他 7 個詞彙

可在 `app.js` 的 `wordDatabase` 陣列中編輯或新增單字。

## 🎨 技術亮點

- **CSS 3D Transform**：使用 `perspective` 和 `rotateY` 實現逼真的翻頁效果
- **Keyframe 動畫**：頁面加載時的淡入動畫
- **漸變背景**：現代化的藍紫色漸變設計
- **語義化 HTML**：結構清晰易懂
- **Vanilla JavaScript**：無框架依賴，代碼輕量高效

## 🔧 自訂方法

### 新增單字
編輯 `app.js`，在 `wordDatabase` 陣列中添加新物件：

```javascript
{
    word: '單字',
    translation: '翻譯',
    pos: '詞性',
    phonetic: '/音標/'
}
```

### 修改顏色主題
編輯 `style.css` 中的 `:root` 變數：

```css
:root {
    --primary-color: #1e293b;
    --accent-color: #3b82f6;
    /* ... */
}
```
