# 📚 背單字卡片 - Flashcard App（升級版）

一個現代化的英文單字學習應用，採用互動式翻頁卡片設計。內建 **API 自動填入功能**，只需輸入英文單字，自動從 API 獲取翻譯、詞性、音標、例句等資訊。

## ✨ 主要功能

### 🎴 學習模式
- **3D 翻頁效果**：點擊卡片或按鈕可產生流暢的翻頁動畫
- **完整單字資訊**：
  - 正面：英文單字
  - 背面：中文翻譯、詞性、音標、例句、字根分析
- **鍵盤快捷鍵**：
  - `←` 上一個單字
  - `→` 下一個單字
  - `Space` 翻轉卡片
- **進度顯示**：實時顯示當前位置（X / 總數）
- **循環播放**：到達最後一個單字後會循環回到第一個

### 📝 管理模式
- **表單新增**：手動輸入單字資訊
- **🔄 自動填入按鈕**：
  - 只需輸入英文單字，自動調用 **Free Dictionary API**
  - 自動填充：
    - ✅ 中文翻譯（英文定義）
    - ✅ 詞性（noun/verb/adjective 等）
    - ✅ 音標（國際音標）
    - ✅ 例句
  - 可手動編輯任何欄位
- **單字列表**：查看、刪除已新增的單字
- **本地存儲**：所有單字自動保存到瀏覽器本地存儲

## 📖 操作說明

### 學習模式

| 操作 | 效果 |
|------|------|
| 點擊卡片中央 | 翻轉卡片 |
| 按「🔄 翻轉卡片」按鈕 | 翻轉卡片 |
| 按「← 上一個」按鈕 | 切換到上一個單字 |
| 按「下一個 →」按鈕 | 切換到下一個單字 |
| 按左箭頭鍵 `←` | 切換到上一個單字 |
| 按右箭頭鍵 `→` | 切換到下一個單字 |
| 按空白鍵 `Space` | 翻轉卡片 |

### 管理模式

1. **自動填入流程**：
   - 在「英文單字」欄位輸入單字（如 `tomato`）
   - 點擊「🔄 自動填入」按鈕
   - 等待 API 響應（通常 1-2 秒）
   - 所有欄位自動填充
   - 可選擇編輯任何欄位

2. **手動新增流程**：
   - 填入「英文單字」和「中文翻譯」（必填）
   - 選擇填入其他資訊（可選）
   - 點擊「✅ 新增單字」

3. **刪除單字**：
   - 在「已有單字」列表中點擊「刪除」按鈕

## 🔌 API 說明

應用使用 **Free Dictionary API**：
- **API 端點**：`https://api.dictionaryapi.dev/api/v2/entries/english/{word}`
- **功能**：免費獲取英文單字的定義、詞性、音標、例句等
- **無需認證**：無需 API Key，直接使用

### 自動填入欄位對應

| 表單欄位 | API 來源 |
|---------|---------|
| 中文翻譯 | 第一個定義 (definition) |
| 詞性 | partOfSpeech |
| 音標 | phonetic |
| 例句 | 定義中的 example 欄位 |
| 字根分析 | 無（需手動填寫） |

## 💾 數據存儲

- **存儲位置**：瀏覽器 LocalStorage
- **存儲鍵名**：`flashcardWords`
- **數據格式**：JSON 陣列
- **刪除數據**：
  1. 打開瀏覽器開發者工具（F12）
  2. 進入 Console 標籤
  3. 執行：`localStorage.removeItem('flashcardWords')`
  4. 或清除所有本地存儲：`localStorage.clear()`

## 🎨 技術亮點

- **CSS 3D Transform**：使用 `perspective` 和 `rotateY` 實現逼真的翻頁效果
- **Fetch API**：非同步調用外部 API 獲取數據
- **LocalStorage API**：本地數據持久化
- **Vanilla JavaScript**：無框架依賴，代碼輕量高效
- **Responsive Design**：適配所有屏幕尺寸

## 🔧 自訂方法

### 新增預設單字

編輯 `app.js`，在 `loadWordsFromStorage()` 函數中修改 `wordDatabase` 初值：

```javascript
wordDatabase = [
    {
        word: '單字',
        translation: '翻譯',
        pos: '詞性',
        phonetic: '/音標/',
        example: '例句',
        etymology: '字根分析'
    }
];
```

### 修改顏色主題

編輯 `style.css` 中的 `:root` 變數：

```css
:root {
    --primary-color: #1e293b;
    --accent-color: #3b82f6;
    /* ... */
}
```

### 更換 API

如果 Free Dictionary API 不可用，可更換其他 API：
1. 修改 `app.js` 中的 `API_URL` 變數
2. 在 `autoFillWord()` 函數中調整 API 響應解析邏輯

## ⚠️ 已知限制

1. **API 限制**：Free Dictionary API 每小時有請求次數限制，若超過可能被暫時限制
2. **例句語言**：API 返回的例句為英文，需手動翻譯或補充
3. **字根分析**：API 無提供此功能，需用戶手動填寫
